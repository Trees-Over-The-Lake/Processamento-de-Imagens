\begin{abstract}
    O presente trabalho descreve as técnicas implementadas para o desenvolvimento de 
    um classificador de imagens de radiografia de mama com o fim de construir um 
    classificador que possa ser utilizado para identificar a qual classe \emph{BI-RADS} 
    uma imagem de exame pertence. 

    O trabalho implementa técnicas de pré-processamento de imagens e descritores 
    de textura para encontrar características que auxiliam na identificação bem-sucedida 
    da classe \emph{BI-RADS} de uma imagem. Para o fim de classificar a imagem, é 
    implementada uma \emph{Support Vector Machine} de \emph{kernel} linear que recebe as imagens 
    resultantes do pré-processamento. Com a implementação de ambas as técnicas foi utilizado um 
    conjunto de 400 imagens para treinamento e avaliação do modelo, sendo obtida uma acurácia de aproximadamente 65\%.    
\end{abstract}
